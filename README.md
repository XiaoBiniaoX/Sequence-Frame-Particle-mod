Sequence-Frame-Particle-mod

Sequence Frame Particle Player is a Forge 1.20.1 mod that allows players to play custom frame-by-frame animations using commands, in a style similar to particle effects.

Type /bin in the chat, then press Tab to see the available command syntax.

This mod makes it easy to create particle effects. The particle images must be in an n×n format, and you can add unlimited frame images through resource packs.

If you like, you can use commands to set the delay or FPS for particle playback. In a sense, this lets you play animations inside the game.

You can also configure directional movement, making particles fly toward a specific location.

Place your sequence-frame textures in:

resourcepacks\your_pack\assets\xulielizimod\textures\particle
